The maximum value which can be spent through all the framework agreements announced in this notice, including options and renewals of contracts.<br/><br/>Additional information:<br/><br/>The value provided is a threshold value that implicity means that it cannot be exceeded however it may not be reached during the execution of a contract.<br/><br/>The Framework Agreements in a CAN are to be traced back and added to provide this value.
