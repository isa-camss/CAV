Class   		Name:  
  		Name: epo:hasEstimatedValue
 
Details:
A forecast of the value of the procurement before competition.<br/><br/>Additional Information<br/>Different cases of estimated values may refer to a lot, the global value of the procedure, or of a combinatorial value of a group of lots.<br/>The forecast is calculated by the buyer and covers all revenues  whether coming from the buyer or third parties.<br/>See for example recital (19), Article 5 of Directive 2014/24/EU and other articles from the rest of Directives about procurement.<br/><br/>In the case of framework agreements and dynamic purchasing systems this refers to the maximum estimated value.<br/><b><br/></b><b>WG Approval 05/12/2019</b><br/><br/>

Details:
The value of all lots awarded for this notice, including options and renewals. <br/><br/>Additional information:<br/>The values of the individual Lots awarded under a framework agreement and mentioned in this notice are included.<br/><br/>The values of the individual lots announced in a CAN are to be traced back and added to provide this value.<br/><br/>The path to get the value of an individual Lot is:<br/><br/>CAN isA Notice ,<br/>CAN notifiesAwardResult AwardDecision ,<br/>AwardDecision refersTo Lot ,<br/>hasAwardedValue ProcurementValue .<br/><br/>WG Approval 03/12/2019<br/><br/>

epo:Lot
Class   		Name:  
  		Name: epo:hasEstimatedBuyerConcessionRevenue
 
Details:
The expected payments made by the buyer to the economic operator awarded the concession that are not directly related to the use of the concession.<br/><br/><u>Additional Information:</u><br/>For example the public buyer pays a yearly fee to provide a ticketing solution to the public.  The fee the public pays for every ticket sold through the solution is not included in this estimation but in the estimation of the user concession revenue.<br/><br/>WG Approval 07/01/2020<br/>
epo:ContractAwardNotice
Class   		Name:  
  		Name: epo:hasTotalFrameworkValue
 
Details:
The maximum value which can be spent through all the framework agreements announced in this notice, including options and renewals of contracts.<br/><br/>Additional information:<br/><br/>The value provided is a threshold value that implicity means that it cannot be exceeded however it may not be reached during the execution of a contract.<br/><br/>The Framework Agreements in a CAN are to be traced back and added to provide this value. The path to get the value of an individual Framework Agreement is:<br/><br/>CAN isA Notice,<br/>CAN notifiesAwardResult AwardDecision,<br/>AwardDecision refersTo Lot,<br/>Lot uses Technique, <br/>FrameworkAgreement isA Technique,<br/>FrameworkAgreement hasAwardedValue ProcurementValue .<br/><br/>WG Approval 03/12/2019<br/><br/><br/>
